# MaxKB

## 一句话总结

企业级智能体平台，适合知识库问答、客服与复杂业务流程编排。

## 它解决什么问题

- 企业做知识库问答时，通常要在 RAG、工作流和工具调用之间自己拼装
- 客服和内部知识助手往往需要零代码或低代码接入现有系统
- 团队希望私有化部署企业 agent，而不是全部依赖外部托管服务

## 适合谁

- 企业知识库和客服团队
- 想私有化部署 agent 的组织
- 需要把 RAG 和业务流程结合起来的团队

## 核心能力

- RAG 知识库：支持文档上传和在线抓取
- Agent 工作流：适合复杂业务场景编排
- 系统集成：可快速接入第三方业务系统

## 典型使用场景

- 企业客服知识助手
- 内部知识问答和培训支持
- 复杂业务流程的智能化改造

## 为什么值得关注

- 不是单点功能，而是比较完整的企业 agent 平台
- 客服和知识管理场景都能直接受益
- 对要做私有化 AI 应用的团队很有吸引力
