# Atomic

## 一句话总结

一个把 Markdown 笔记自动转成语义知识图谱的个人知识库系统，支持向量搜索、Wiki 合成、图谱画布和 AI 对话。

## 它解决什么问题

- Markdown 笔记越积越多后，关联关系很难靠人工维护
- 传统关键词搜索无法很好找出语义相近的知识点
- 想把零散笔记进一步变成可问答、可浏览、可生成文章的知识库

## 适合谁

- Markdown 笔记重度用户
- 想构建个人知识库和第二大脑的人
- 需要把笔记与 AI 检索能力结合起来的知识工作者

## 核心能力

- 原子化笔记：自动切块、嵌入、打标签并按语义关联
- 语义搜索：基于向量检索而不是仅靠关键词
- Wiki 合成：从笔记中生成带引用的主题文章
- 画布浏览：通过图谱关系可视化知识连接
- Agentic Chat：对知识库做 RAG 式对话查询

## 典型使用场景

- 把多年 Markdown 笔记重新组织成可检索的知识网络
- 从零散笔记中自动合成某个主题的 wiki 页面
- 用语义搜索快速找出相关资料，而不是手工翻目录

## 为什么值得关注

- 它把“笔记存储”升级成“知识图谱 + AI 检索”系统
- 对已经有 Markdown 习惯的人，上手门槛相对低
- 支持桌面、Web、浏览器扩展和 MCP，扩展性比较强
