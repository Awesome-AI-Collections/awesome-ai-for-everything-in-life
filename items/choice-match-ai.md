# choice-match-ai

## 一句话总结

choice-match-ai 是一个时尚电商推荐引擎，可根据场景、预算、尺码和偏好解析，输出更个性化的搭配建议。

## 它解决什么问题

- 服饰电商用户需求表达复杂，普通筛选器经常不够用
- 商品推荐如果只看关键词，很难覆盖场景和搭配关系
- 商家希望把“像导购一样推荐”引入站内体验

## 适合谁

- 做服饰和时尚电商的团队
- 想增强个性化推荐的产品团队
- 需要偏好解析和推荐 API 的创业项目

## 核心能力

- 偏好解析：理解预算、场景、尺码和气候等条件
- 搭配推荐：输出更接近真实购物决策的组合建议
- API 化：适合接入电商站点和推荐服务

## 典型使用场景

- 服饰导购和搭配推荐
- 个性化商品发现
- 对话式购物推荐原型

## 为什么值得关注

- 业务目标明确，直接服务电商推荐
- 同时结合 NLP 与推荐逻辑，比单纯规则筛选更贴近真实需求
- 很适合观察 AI 导购在垂直品类里的落地方式
