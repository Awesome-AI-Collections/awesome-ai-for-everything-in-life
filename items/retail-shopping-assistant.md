# Retail Shopping Assistant

## 一句话总结

Retail Shopping Assistant 是 NVIDIA 的零售购物顾问蓝图，用多 agent 架构把商品搜索、图像检索和购物车交互组合成完整购物体验。

## 它解决什么问题

- 零售购物助手往往只会聊天，缺少真正可执行的购物流程。
- 商品搜索、图像检索和购物车操作如果分散，体验很难闭环。
- 团队想做高质量购物顾问时，缺少工程化蓝图。

## 适合谁

- 做零售电商 AI 体验的团队
- 想搭建多 agent 购物顾问的产品与工程团队
- 需要图片搜商品和购物车交互的互联网业务团队

## 核心能力

- 多 agent 编排：把购物任务拆成可协作的智能组件。
- 图像搜索：支持基于视觉线索找商品。
- 购物车交互：不仅回答问题，还能推进购物流程。

## 典型使用场景

- 零售购物顾问 demo
- 商品咨询与购物车协同体验
- 视觉搜索驱动的电商导购

## 为什么值得关注

- 它不是单纯 prompt 示例，而是更完整的零售蓝图。
- 对需要工程参考的团队来说，价值比轻量 demo 更高。
- 能很好补充现有电商 AI 条目的“系统化架构”维度。
