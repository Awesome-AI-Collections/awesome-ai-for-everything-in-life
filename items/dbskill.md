# dbskill

## 一句话总结

一个把商业诊断、内容优化、标题公式和 AI 写作检测做成 Claude Code skills 的开源工具箱。

## 它解决什么问题

- 很多创业者和内容团队知道“要诊断”，但缺少可复用的方法论工具箱
- 商业判断、对标、内容开头、标题优化和执行卡点常被分散在不同工具里
- 想把个人方法论沉淀为 AI 可调用资产时，往往缺少现成结构样板

## 适合谁

- 做内容增长、商业诊断和账号运营的人
- 想把方法论知识包接入 Claude Code 或自建 RAG 的实践者
- 需要诊断标题、开头、执行力和 AI 味内容的个体创业者

## 核心能力

- 主入口路由：用 `/dbs` 作为统一入口，把问题导向不同 skill
- 商业诊断工具箱：覆盖商业模式诊断、对标分析、概念拆解和执行力诊断
- 内容优化能力：包含短视频开头、标题公式、AI 写作检测等高频内容场景
- 知识库开放：既提供方法论文档，也提供结构化原子库，便于二次开发
- 安装路径明确：支持通过 Claude Code plugin marketplace 或 `npx skills add` 接入

## 典型使用场景

- 给创作者或创业者做商业模式与内容策略诊断
- 在写作或运营流程里检查“AI 味”、优化标题和开头
- 把原子化知识点导入向量数据库，做商业顾问型 RAG

## 为什么值得关注

- 它不只是 skill 壳子，还把知识包、原子库和工作流关系都整理得比较完整
- README 里已经把不同 skill 的职责边界和联动路径说清楚了
- 对想把个人经验产品化为 Agent 能力的人来说，很有参考价值
