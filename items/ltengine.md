# LTEngine

## 一句话总结

一个本地可自托管的 AI 机器翻译 API，兼容 LibreTranslate，适合多语业务和国际化流程。

## 它解决什么问题

- 团队需要本地化翻译能力，但不想完全依赖外部 SaaS
- 国际化流程里，翻译常常需要 API 化和可控部署
- 一些业务需要兼顾多语言支持和数据可控性

## 适合谁

- 做国际化产品和多语内容的团队
- 关注本地部署翻译 API 的开发者
- 对翻译成本和数据可控性敏感的企业

## 核心能力

- 自托管翻译 API：方便接进业务系统和自动化流程
- 本地 AI 翻译：更适合有隐私和成本要求的场景
- LibreTranslate 兼容：更利于迁移已有工作流

## 典型使用场景

- 多语言业务后台翻译
- 国际化内容流水线
- 企业内部翻译服务化

## 为什么值得关注

- 对需要“可控翻译基础设施”的团队很有价值
- 相比只做客户端翻译，更适合系统级接入
- 在本地 LLM 翻译路线里，定位比较清晰
