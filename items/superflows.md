# Superflows

## 一句话总结

一个把 AI 助手直接嵌入业务软件的开源平台，能调用产品 API 回答问题、执行操作和分析数据，适合 CRM、客服、运营后台等内部系统。

## 它解决什么问题

- 员工明明在 CRM、运营后台或客服系统里就有数据，却还要手工翻页面、导报表和点很多步骤
- 很多团队想做“会做事”的 AI 助手，而不只是一个只能聊天的问答框
- 把 AI 接进内部系统时，经常卡在 API 对接、权限控制和前端集成上

## 适合谁

- 外贸、销售、客服和运营团队的内部工具建设者
- 想给 SaaS 产品或内部系统加 AI Copilot 的产品与技术团队
- 需要把业务动作和数据分析交给 AI 辅助完成的组织

## 核心能力

- API 动作执行：让 AI 通过调用业务系统 API 完成查询和操作
- 数据分析与绘图：支持写代码分析数据并生成图表
- RAG 与嵌入式 UI：可结合静态知识检索回答问题，并提供现成 React 组件快速接入

## 典型使用场景

- 在 CRM 里直接问“上月哪些线索转化率最高”，并让 AI 调接口补全答案
- 销售主管批量调整客户跟进动作，而不是逐个改记录
- 给内部运营后台接入一个能查数、能执行任务的 AI 助手

## 为什么值得关注

- 它强调的是“把 AI 放进现有业务软件”，不是另起一个独立聊天工具
- 同时覆盖问答、任务执行和数据分析，贴近企业真实工作流
- 对外贸、互联网公司的运营型岗位非常实用
