# Tiledesk Design Studio

## 一句话总结

一个面向客服与业务对话自动化的可视化无代码设计器，把 LLM 对话、流程图和自动化编排放到同一个界面。

## 它解决什么问题

- 业务团队想搭 AI 客服或业务对话流程，但不想全靠工程团队写代码
- 单纯聊天机器人很难承载复杂分支和自动化动作
- 运营团队需要一个既能设计对话，又能快速迭代流程的工具

## 适合谁

- 客服运营和支持团队
- 需要搭建企业聊天机器人和会话自动化的运营负责人
- 想用可视化方式管理 LLM 对话流程的团队

## 核心能力

- 无代码设计：用图形化方式设计会话和自动化
- LLM 对话编排：把 GPT / LLM 能力接进对话流程图
- 运营可控：比纯代码 bot framework 更适合业务侧持续迭代

## 典型使用场景

- 客服机器人和售前咨询自动化
- FAQ、工单分流和多步骤业务对话
- 给中小团队快速搭建“可改、可测、可运营”的 AI 会话流程

## 为什么值得关注

- 明确定位为 Voiceflow / Botpress 替代方案，产品边界清晰
- 同时覆盖会话设计和自动化，对互联网业务支持团队很实用
- 开源形态适合自托管或深度二次开发
