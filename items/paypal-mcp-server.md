# PayPal MCP Server

## 一句话总结

一个把 PayPal 发票、订单、退款、争议和订阅动作开放给 AI 助手的 MCP server，适合跨境收款、售后和财务协作。

## 它解决什么问题

- 外贸和跨境团队经常要在 PayPal 后台来回切换查发票、退款和订单状态。
- 客服、财务、运营之间涉及 PayPal 时，沟通链条长，处理速度慢。
- 想让 AI 助手直接帮忙查支付和争议情况，但缺一个标准工具层。

## 适合谁

- 外贸收款团队
- 跨境电商运营
- 财务与售后团队

## 核心能力

- 发票管理：支持创建、查询、发送和催款。
- 支付动作：支持订单创建、支付和退款。
- 风险售后：支持争议列表和争议详情查询。
- 订阅与产品：支持产品、订阅计划和订阅管理。

## 典型使用场景

- 让 AI 助手回答“这笔 PayPal 订单有没有付款成功”。
- 用 AI 帮客服发起退款或查看退款状态。
- 财务让 AI 汇总近期交易、发票和争议情况。

## 为什么值得关注

- 这是 PayPal 官方组织下的 MCP 项目，可信度更高。
- README 工具覆盖面很全，不只是查数据，还能执行核心支付动作。
- 对跨境收款、售后退款和订阅业务都很实用。
