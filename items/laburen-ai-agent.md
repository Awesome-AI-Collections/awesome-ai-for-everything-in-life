# Laburen AI Sales Agent

## 一句话总结

一个接入 WhatsApp 的电商销售代理，可自动推荐商品、管理购物车并推进客户咨询流程。

## 它解决什么问题

- 电商团队在 WhatsApp 上接待顾客时，很多重复问答难以规模化处理
- 用户问到商品推荐或购物车问题时，人工客服响应速度容易成为瓶颈
- 团队想把对话式销售流程和商品数据直接接起来

## 适合谁

- 使用 WhatsApp 做获客和转化的跨境电商团队
- 想做 AI 商品推荐和私域咨询自动化的卖家
- 需要验证“聊天即销售”流程的创业团队

## 核心能力

- WhatsApp 销售对话：通过 Twilio / WhatsApp API 接入真实聊天渠道
- 商品推荐：结合 Gemini 处理用户意图并输出商品建议
- 购物车协助：在聊天里衔接购物车和购买前咨询

## 典型使用场景

- 海外私域渠道的商品推荐和答疑
- 跨境电商的聊天式销售前置接待
- 把 FAQ、推荐和购物车推进整合成一个自动化流程

## 为什么值得关注

- 场景非常具体，直接落在跨境电商和 WhatsApp 销售
- 不是纯 demo，提供了生产环境、调试和 API 文档入口
- 适合研究“聊天渠道 + 商品推荐 + 转化动作”的组合
