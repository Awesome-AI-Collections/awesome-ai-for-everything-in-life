# Eigent

## 一句话总结

一个开源 Cowork 桌面应用，可用多 Agent 协作把复杂办公流程变成自动化任务，适合需要跨网页、文档和工具执行任务的团队。

## 它解决什么问题

- 很多工作任务不是单一步骤，而是要查网页、写文档、调工具、回消息一起做
- 团队想用多 Agent 自动化办公，但又不想完全依赖闭源云产品
- 企业希望对模型、权限和本地部署有更强控制力

## 适合谁

- 需要把重复办公流程自动化的运营和支持团队
- 想试用多 Agent 协作的业务团队
- 对本地部署和隐私有要求的组织

## 核心能力

- 多 Agent 协同：多个不同角色的 Agent 并行处理复杂任务
- MCP 工具接入：可连接网页、文档、Slack、Google 套件等工具
- 本地部署：支持本地运行和自定义模型

## 典型使用场景

- 自动整理资料、生成报告并同步到团队沟通工具
- 为运营团队搭建跨工具执行的 AI 办公助手
- 用多 Agent 拆分并执行较长链路的研究或行政任务

## 为什么值得关注

- 它强调的是“AI 同事桌面”，而不是单一聊天窗口
- 多 Agent 和 MCP 结合后，对复杂办公流更有想象空间
- 开源和本地部署让试点成本更低、控制力更强
