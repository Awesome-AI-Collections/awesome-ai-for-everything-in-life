# OpenBB

## 一句话总结

一个面向分析师、量化和财务团队的开源金融数据平台，可把多源数据接到 Python、Excel、Workspace、REST API 和 AI 代理中统一使用。

## 它解决什么问题

- 财务和研究团队经常要在多个数据源、Excel、脚本和图表工具之间来回切换
- 很多组织想把专有数据、公开数据和 AI 分析工具接到同一条工作流里
- 研究输出常常难以同时服务人类分析师、报表系统和 AI 代理

## 适合谁

- 外贸企业的财务、投融资、经营分析和战略团队
- 需要做市场、证券、宏观或行业研究的分析人员
- 想把金融数据统一接入 AI Copilot 和研究仪表盘的组织

## 核心能力

- 多源金融数据接入：整合专有、授权和公开数据供下游统一调用
- 多终端消费：支持 Python、Excel、OpenBB Workspace、REST API 和 AI agents
- 研究与分析基础设施：适合搭建研究面板、分析流程和财务 AI 应用

## 典型使用场景

- 财务分析师把经营和外部市场数据统一拉进 Excel 或 Workspace 分析
- 研究团队用 Python 和 API 快速复用同一套数据源
- 给内部 AI 代理接入金融与经营数据，做问答和辅助分析

## 为什么值得关注

- 它强调“connect once, consume everywhere”，很适合组织级数据分发
- 对财务、研究和量化团队都很直接，不局限于纯开发者场景
- 能把传统分析工具链和 AI 代理工作流连起来
