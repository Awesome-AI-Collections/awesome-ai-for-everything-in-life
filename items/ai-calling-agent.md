# AI Calling Agent

## 一句话总结

AI Calling Agent 把 OpenAI Realtime API 和 Twilio Voice 连接起来，可用于客服热线、回访外呼和自动化电话沟通。

## 它解决什么问题

- 电话场景很难像网页聊天那样快速接入 AI
- 客服和销售外呼经常需要 7x24 响应或低成本筛选
- 团队想验证语音 Agent 是否能承担部分来电和外呼任务

## 适合谁

- 有电话客服和回访需求的团队
- 做销售外呼自动化试点的团队
- 需要实时语音交互方案的产品团队

## 核心能力

- 实时语音：低延迟处理来电与对话
- 电话集成：通过 Twilio 快速接入真实呼叫链路
- 场景可配：可改 prompt、语音风格和工作流逻辑

## 典型使用场景

- 客服热线的基础问答分流
- 线索回访和预约确认外呼
- 合规提醒或通知型电话机器人

## 为什么值得关注

- 不是纯语音 demo，而是带真实电话集成
- 对销售和客服团队都能产生直接业务价值
- 很适合研究语音 Agent 在互联网业务里的可行边界
