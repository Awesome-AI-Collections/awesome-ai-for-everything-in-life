# geo-seo-claude

## 一句话总结

一个面向 AI 搜索时代 GEO 与 SEO 审计的 Claude Code 工具，可并行分析网站可引用性、内容质量和技术基础。

## 核心场景

- 对网站做 GEO + SEO 全面体检
- 生成面向客户交付的 Markdown 或 PDF 审计报告
- 提前优化 ChatGPT、Claude、Perplexity 等 AI 搜索可见度

## 为什么值得关注

- 把 GEO 这类新需求做成了可执行审计流程
- 适合内容团队、SEO 团队和代理服务商
