# Tiledesk

## 一句话总结

一个开源客服与 Agent 平台，支持 LLM 对话、人工接管和多渠道工作流编排。

## 它解决什么问题

- 客服自动化往往缺少和人工协作的平衡点
- 团队需要多渠道客服平台，但又想把 LLM Agent 接进去
- 只做聊天机器人不够，实际业务还需要流程和接管机制

## 适合谁

- 想做客服自动化的团队
- 需要 AI 与人工协同的支持团队
- 研究开源客服平台的人

## 核心能力

- 开源客服平台：覆盖对话、工单和支持流程
- LLM Agent：可接入更智能的自动回复和任务处理
- 人工接管：在复杂场景下切回人工支持

## 典型使用场景

- 多渠道客服自动化
- 电商和 SaaS 客服协作
- AI 客服平台原型和二次开发

## 为什么值得关注

- 它处理的是客服落地中的关键问题，不只是“能不能回答”
- 人工接管能力对生产场景非常关键
- 对客服团队和 Agent 产品团队都很有参考价值
