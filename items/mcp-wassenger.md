# MCP Wassenger

## 一句话总结

一个把 Wassenger WhatsApp API 接入 AI 助手的 MCP connector，支持消息发送、聊天总结、群组管理、广播和 campaign 自动化。

## 它解决什么问题

- 外贸销售和客服大量工作发生在 WhatsApp，但普通 AI 助手接不进去。
- 团队想让 AI 帮忙回消息、做聊天总结、跑广播活动时，经常只能手工操作。
- 客户沟通、群组管理、消息调度和活动投放缺一个统一连接层。

## 适合谁

- 外贸销售
- WhatsApp 客服团队
- 需要 AI 管理客户消息的人

## 核心能力

- 消息与会话：支持消息发送、回复、搜索和聊天总结。
- 联系人与群组：支持联系人、群组和标签管理。
- 自动化：支持消息调度、自动回复和 workflow。
- 运营活动：支持广播和 campaign 相关操作。

## 典型使用场景

- 让 AI 按客户姓名或号码发送 WhatsApp 跟进消息。
- 把当天客户对话自动总结给销售或主管。
- 跑一批 WhatsApp 广播活动，查看投递状态和失败消息。

## 为什么值得关注

- WhatsApp 是外贸销售和跨境客服极常见的主渠道，这类 MCP 很实用。
- README 给了很多真实 prompt 和自动化示例，不只是技术样板。
- 它把消息发送、总结、群组管理和活动运营放到了同一个 AI 接口里。
