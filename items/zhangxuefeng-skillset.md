# Zhangxuefeng Skillset

## 一句话总结

一个面向高考志愿填报与专业选择的开源 AI 知识库，可作为 OpenClaw skill、微信顾问入口或 RAG 教育问答底座。

## 它解决什么问题

- 很多家庭在高考志愿填报上信息不对称，难以系统比较专业、学校、城市与就业前景
- 传统分数匹配工具只给“能报什么”，但缺少完整决策框架
- 教育咨询知识往往分散在视频、直播和个人经验里，不易沉淀成可复用知识库

## 适合谁

- 需要高考志愿填报辅助的学生与家长
- 想把升学咨询知识库接进微信、企业微信或 RAG 系统的人
- 关注 AI 在教育辅助与决策支持场景落地的人

## 核心能力

- 决策知识库：围绕专业选择、志愿策略、就业前景、院校选择和大学规划构建模块化内容
- AI 时代校正：把 2026-2030 视角下的专业前景与 AI 冲击因素纳入判断框架
- Skill 化部署：可直接作为 OpenClaw skill 安装，在 Claude Code 或 OpenClaw 体系里调用
- 多入口复用：既可直接阅读 Markdown，也可导入 Dify、FastGPT、RagFlow 等 RAG 平台
- 微信接入思路：README 明确把微信、企业微信和 QQ 作为典型触达入口

## 典型使用场景

- 给考生家庭提供结构化的志愿填报对话顾问
- 将教育咨询知识内容做成 RAG 问答机器人
- 用现成的知识框架快速搭建一个“高考规划助手”原型

## 为什么值得关注

- 这不是单纯的院校分数匹配工具，而是更强调决策顺序和判断逻辑
- README 对知识库结构、部署方式和使用路径写得很完整，上手成本不高
- 它把教育咨询场景和 AI skill / RAG 落地方式结合得比较直接，适合做行业化案例参考
