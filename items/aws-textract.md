# AWS Textract

## 一句话总结

AWS 的文档文本提取服务，适合发票、装箱单等业务单证识别。

## 核心场景

- 提取发票、装箱单和扫描件中的文字
- 把文本数据送入后续系统或工作流

## 为什么值得关注

- 适合单证自动化处理
- 和 AWS 体系结合比较方便
