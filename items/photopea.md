# Photopea

## 一句话总结

一个浏览器内可直接使用的在线图像编辑器，支持 PSD、AI、Sketch、PDF 等格式，适合快速改图、抠图和跨格式素材处理。

## 它解决什么问题

- 很多人只想快速改图或导出素材，不想装体积很重的桌面设计软件
- 收到 PSD、AI、Sketch、PDF 等不同格式文件时，兼容性经常成为门槛
- 轻量设计、简单抠图和素材修订常常只需要网页工具就能完成

## 适合谁

- 需要临时改图、裁图、导出素材的内容和运营团队
- 没有 Photoshop 但要处理 PSD 或 AI 文件的个人用户
- 需要在低配置设备或浏览器环境里完成图像编辑的人

## 核心能力

- 多格式支持：支持 PSD、AI、Sketch、PDF、SVG 以及常见位图格式
- 在线编辑：直接在浏览器完成图层编辑、导出和格式转换
- 轻量替代：适合做快速修图、素材尺寸调整和简单设计处理

## 典型使用场景

- 打开客户或设计师给的 PSD、AI 文件并做小修改
- 处理社媒封面、电商主图和简单抠图需求
- 在没有本地设计软件的环境里做素材导出和转换

## 为什么值得关注

- 对多种设计文件格式的支持很广，临时处理文件很方便
- 浏览器即开即用，适合碎片化设计和内容团队协作
- 官方 GitHub 虽然不是完整开源代码仓库，但可以作为问题反馈和更新观察入口

## 类似项目

- [Canva Photo Editor](https://www.canva.com/photo-editor/) - 更偏模板化和协作化的在线设计体验
- [Adobe Photoshop](https://www.adobe.com/products/photoshop.html) - 功能更重更全，但成本和门槛也更高
