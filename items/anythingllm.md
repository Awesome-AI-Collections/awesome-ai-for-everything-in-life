# AnythingLLM

## 一句话总结

一个可私有部署的开源 AI 工作空间，支持文档对话、AI Agent、多用户权限和 MCP 扩展，适合团队搭建内部 AI 助手。

## 它解决什么问题

- 团队想要“能聊天又能接文档和工具”的内部 AI 工作台，但现成 SaaS 不一定能私有化
- 多人共用 AI 时，需要权限控制、数据隔离和统一模型接入
- 很多团队不只想做文档问答，还想做 Agent、嵌入网站和后续集成

## 适合谁

- 需要自托管内部 AI 助手的运营、支持和知识管理团队
- 想把文档、知识库和 Agent 放到同一工作空间里的组织
- 希望快速接入多种 LLM、向量库和语音能力的团队

## 核心能力

- 文档对话与工作空间：支持上传多种文档并在工作空间内持续对话
- Agent 与扩展：支持无代码 Agent builder、MCP 兼容和开发 API
- 团队化部署：支持多用户、权限控制、嵌入式聊天组件和多模型接入

## 典型使用场景

- 为内部运营和客服团队搭建共享知识助手
- 将产品手册、SOP 和 FAQ 接入统一 AI 工作台
- 给官网或帮助中心嵌入可控的聊天入口

## 为什么值得关注

- 它把文档对话、Agent、嵌入和多用户能力做在同一个产品里
- 本地运行和自托管门槛相对低，适合先试点再扩展
- LLM、向量库和语音支持面很广，方便不同团队按现有栈接入

## 类似项目

- [Open WebUI](open-webui.md) - 更偏模型统一入口和团队聊天界面
- [LibreChat](librechat.md) - 同样支持多模型接入和团队化 AI 工作台
