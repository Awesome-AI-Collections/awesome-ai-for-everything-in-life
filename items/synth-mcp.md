# Synth MCP

## 一句话总结

一个接入 Synth Finance API 的远程 MCP server，可提供实时与历史汇率、外汇数据和更多金融信息，适合跨境业务做 FX 参考与走势查询。

## 它解决什么问题

- 有些跨境业务不只要单次换算，还想看汇率走势和历史区间。
- 财务、运营和老板问汇率时，希望 AI 能给更完整的上下文。
- 单纯的换算工具不够覆盖“实时 + 历史 + 解释”这种复合需求。

## 适合谁

- 外贸运营团队
- 财务与结算人员
- 需要更完整 FX 数据参考的人

## 核心能力

- 实时汇率：支持 live forex 查询。
- 历史汇率：支持按过去日期取汇率。
- 更丰富金融数据：除了汇率，还能扩展到更多金融信息场景。

## 典型使用场景

- 在结算前查看当前汇率和近期变化。
- 做多币种收入、成本和利润分析。
- 给 AI 助手补充更完整的 FX 背景信息。

## 为什么值得关注

- 比简单汇率换算更丰富，适合分析型场景。
- README 明确支持实时和历史 FX 数据。
- 对跨境结算和价格波动分析更有帮助。
