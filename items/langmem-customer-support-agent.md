# LangMem Customer Support Agent

## 一句话总结

这是一个带长期记忆能力的客服 Agent，可自动分类咨询、创建工单并在多轮对话中保留用户上下文。

## 它解决什么问题

- 客服机器人常常只记住单轮对话，体验割裂
- 重复客户来询时，团队希望 AI 能记住背景和历史
- 需要把问答和工单衔接起来，减少人工重复录入

## 适合谁

- 做客户支持自动化的团队
- 需要长会话上下文的 SaaS 支持团队
- 想研究 memory 在客服场景价值的产品团队

## 核心能力

- 记忆支持：在连续服务中保留用户上下文
- 工单创建：把无法直接解决的问题转成任务
- 自动分类：帮助支持团队先做问题分流

## 典型使用场景

- 用户多次来询时维持连续服务体验
- 把复杂问题自动落为工单
- 支持团队验证 memory-enhanced agent 的效果

## 为什么值得关注

- 直接切中客服 Agent 常见的“无记忆”短板
- 比泛聊天机器人更接近真实支持流程
- 很适合观察 AI 客服从单轮问答走向持续服务
