# Azure Document Intelligence

## 一句话总结

Azure 的文档智能识别服务，适合企业单证自动提取和结构化。

## 核心场景

- 识别企业文档和票据并做结构化提取
- 把文档处理纳入 Azure 工作流

## 为什么值得关注

- 适合微软云生态下的文档自动化
- 结构化能力适合企业流程接入
