# Intervo

## 一句话总结

一个开源 AI 语音平台，适合电话销售、语音客服和自动呼叫类场景。

## 它解决什么问题

- 企业希望把 AI 语音能力直接接进销售或客服流程
- 纯文本 Agent 解决不了电话和语音会话场景
- 语音场景往往涉及实时性、流程控制和接待质量，落地门槛更高

## 适合谁

- 做电话销售和外呼的团队
- 需要自动语音客服的企业
- 想搭语音 Agent 产品的人

## 核心能力

- AI 语音平台：围绕语音交互而不是纯文本对话构建
- 销售与客服场景：更贴近真实电话触达和服务流程
- 开源可扩展：方便按业务要求做二次开发

## 典型使用场景

- 电话销售外呼
- 售前和售后语音客服
- 语音 Agent 产品验证和集成

## 为什么值得关注

- 直接切进“语音业务流程”这个更难的落地层
- 对销售和客服团队都具备现实价值
- 开源形态便于团队理解其技术和流程结构
