# AI Marketing Skills

## 一句话总结

AI Marketing Skills 是一组面向营销和销售团队的开源 Claude Code skills，适合把内容、增长、市场研究和销售支持流程模板化。

## 它解决什么问题

- 市场和销售团队虽然开始用 AI，但流程经常停留在零散 prompt
- 很多团队需要可复用的工作方法，而不是一次性对话
- 内容生产、竞争研究和外联准备很适合沉淀成标准化 skill

## 适合谁

- 做内容增长的营销团队
- 需要销售支持素材的 GTM 团队
- 想把 AI 工作流标准化的业务团队

## 核心能力

- Skill 模板：把营销任务沉淀为可重复执行的能力单元
- 团队复用：降低每个人重复写 prompt 的成本
- 场景覆盖：适合内容、研究、增长和销售协同

## 典型使用场景

- 内容选题与营销文案生成
- 竞品和市场研究辅助
- 销售准备与 campaign 支持

## 为什么值得关注

- 它强调可复用技能而不是单次 prompt
- 对营销和销售团队的落地价值比纯工具目录更强
- 适合作为业务团队 AI 工作方式的样板
