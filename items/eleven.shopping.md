# eleven.shopping

## 一句话总结

eleven.shopping 是面向 Shopify 店铺的语音购物助手示例，结合 ElevenLabs Agent 与 storefront MCP 做对话式导购。

## 它解决什么问题

- 传统电商搜索和筛选对移动端或语音场景不够友好。
- 很多团队想做语音购物，但缺少可运行的开源样板。
- Shopify 业务如果想快速验证 AI 导购，需要现成 storefront 集成参考。

## 适合谁

- 做 Shopify 独立站的团队
- 想测试语音购物体验的产品团队
- 需要 storefront MCP 示例的开发与运营团队

## 核心能力

- 语音购物：通过自然对话寻找商品。
- Shopify storefront 集成：更贴近真实电商站点。
- MCP 交互参考：适合作为 agent 与店铺前端联动样板。

## 典型使用场景

- 语音导购原型
- Shopify AI 购物助手 demo
- 对话式商品发现体验测试

## 为什么值得关注

- 方向很新，直接把语音 agent 放进购物场景。
- 对出海电商和独立站来说，业务相关性非常强。
- 能补充现有购物 assistant 条目里的“语音 + storefront MCP”维度。
